## Project Overview

This project demonstrates the creation of an interactive business intelligence dashboard using **Power BI**. The dashboard visualizes sales and profit data to help users analyze business performance across different categories, regions, and time periods. The goal of this task is to transform raw data into meaningful visual insights using interactive features such as slicers and filters.

---

## Objective

To design and develop a fully functional interactive dashboard that enables users to explore sales and profit trends and derive actionable insights using visual analytics.

---

## Dataset

The dataset used in this project contains structured sales data including product category, region, sales amount, profit, quantity, and order date. The dataset is loaded into Power BI for visualization and analysis.

---

## Key Fields Used

* **Category**: Product category such as Electronics, Furniture, and Clothing
* **Region**: Sales regions including West, South, East, and North
* **Sales**: Total sales amount
* **Profit**: Profit generated from sales
* **OrderDate / Year**: Time-based field for trend analysis

## Dashboard Components

* **Total Sales Card** – Displays overall sales value
* **Total Profit Card** – Displays overall profit value
* **Sales by Category** – Bar chart comparing sales across categories
* **Profit by Category** – Bar chart comparing profit across categories
* **Profit by Region** – Pie chart showing regional profit contribution
* **Sales Trend by Year** – Line chart showing year-wise sales trend
* **Category Slicer** – Interactive slicer to filter data by product category
* **Region Slicer** – Interactive slicer to filter data by region

* West region contributes the maximum profit among all regions
